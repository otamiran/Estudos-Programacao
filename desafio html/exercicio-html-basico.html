Aqui vamos demonstrar as tags vistas, bem como elaborar uma breve explicação iniciemos com a estrutura básica do html,

Estrutura basica do Html

< !DOCTYPE html >

< html >

< head >

< meta charset="UTF-8"/ >

< title > Document < /title >

< /head >

< body >

-- Conteúdo --

< /body >

< /html >

Com essa estrutura basica de html, você consegue iniciar a elaboração de qualquer pagina usando html, contendo as partes de

  • < title > → Sessão responsávem por alterar a descrição da aba no navegador
  • < head > → Sessão responsável pelo cabeçario da página
  • < body > → Sessão responsável pela composição de todo o conteúdo da página

É muito importante utilizar-se das tags para a elaboração do codigo html, como por exemplo:

  1. < ol > → Lista Ordenada
  2. < ul > → Lista não ordenada
  3. < li > → Item da lista
  4. < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 > → Define a força dos titulos
  5. < p > → Parágrafo
